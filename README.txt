Description
Hotel Reservation System with Object Oriented Paradigm can manipulate Hotel Database with raw offline data which can be imported dan read into the systems. With the object classification in the specified class. The Object Oriented Paradigm simplify program bloc and program size instead of Procedural Programming Paradigm.


Program Reservasi Hotel menggunakan Paradigma Object Oriented programming
spesifikasi :
- bahasa pemrograman : Python 3
- software : Jupyter Notebook

File program :
1. File utama program : Hotel.py
2. File daftar opsi hotel berupa 'rooms.txt' sebagai file penyimpanan data hotel
3. File harga hotel berupa 'price_rooms.txt' sebagai file penyimpan data harga hotel

Ketentuan menjalankan program :
- Nama file rooms dan price_rooms jangan diubah (jika diubah, lakukan rename pada program)
- Ketiga file harus berada pada 1 direktori atau 1 folder yang sama
- Jangan sampai ada file dengan nama dan format yang sama dengan ketiga file tersebut

Cara menjalankan program :

1. Melalui CLI (Command Prompt)
- install python 3 pada OS anda (cari di internet)
- setelah OS anda memiliki Python atau sudah memiliki Python, buka CMD atau terminal
- lakukan perintah perpindahan ke direktori file program berada dengan mengetikkan
"cd <direktori"   
contoh : cd C:\user\system32\folder\

- ketikkan perintah 'Python Hotel.py' pada CMD
- program berhasil dijalankan


2. Melalui jupyter notebook
- install Anaconda3 pada OS anda (cari di internet)
- jupyter notebook akan terinstal otomatis bersamaan dengan Python
- buka Anaconda Prompt (biasanya pada folder Anaconda terinstal)
- ketikkan perintah 'jupyter notebook' pada Anacona Prompt
- Browser anda akan membuka halaman awal Jupyter Notebook
- cari folder file program anda berada, masuk ke dalam folder
- pada sisi kanan atas, pilih File - New - Python3
- anda akan berada pada suatu console pada new tab di browser
- ketikkan 'import Hotel' dan tekan Shift+Enter
- program berhasil dijalankan
